# 타입스크립트 기반 리액트 프로젝트 스토리북 적용

### 스토리북 적용 플러그인

```
@storybook/preset-create-react-app
@storybook/addon-actions
@storybook/addon-links

@storybook/addon-knobs
@storybook/addon-viewport
```

### 실행

* 앱 실행

```
$ npm run start
$ yarn start
```

* 스토리북 테스트

```
$ npm run test
$ yarn test
```

* 스토리북 실행

```
$ npm run storybook
$ yarn storybook
```
